# Vim 配置文件使用说明

## 快速开始

```bash
# 查看详细的配置和使用指南
cat Vim配置使用指南.md

# 复制配置文件到用户目录

# 方式 1: 使用完整版（推荐）
cp vimrc1 ~/.vimrc

# 方式 2: 使用精简版
cp vimrc2 ~/.vimrc

# 编辑配置文件
vim ~/.vimrc

# 重新加载配置
source ~/.vimrc
```

## 配置文件说明

### vimrc1 - 功能完整版
- 完整的 IDE 功能
- 支持 F5 编译运行
- 支持 F8 调试
- 集成多个插件
- 适合日常开发

### vimrc2 - 精简版
- 配置简洁轻量
- 启动速度快
- 适合远程编辑
- 服务器环境优化
